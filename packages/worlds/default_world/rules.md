# 默认世界观规则和禁忌

## 世界规则

### 魔法规则
1. **魔法代价**: 使用魔法会消耗施法者的体力和精神力
2. **学习过程**: 魔法需要学习和练习，不能无师自通
3. **社会态度**: 魔法在某些地方被尊敬，在其他地方被恐惧
4. **限制因素**: 魔法效果受环境、材料和施法者能力限制

### 战斗规则
1. **伤害真实**: 战斗会造成真实的伤害，需要时间恢复
2. **武器装备**: 好的装备提供优势，但不能完全替代技巧
3. **团队配合**: 多人战斗需要配合和策略
4. **逃生可能**: 大多数战斗都有逃跑或求和的选择

### 社会规则
1. **阶级差异**: 不同社会阶级有不同的权利和限制
2. **法律后果**: 违法行为会有相应的法律后果
3. **文化差异**: 不同地区有不同的文化习俗
4. **经济现实**: 金钱和资源是有限的，需要合理管理

## 互动小说规则

### 核心原则
1. **玩家主导**: 玩家的选择应该驱动故事发展
2. **合理后果**: 每个选择都应该有合理的结果
3. **世界连贯**: 世界观必须保持一致和合理
4. **渐进发展**: 角色能力和剧情应该渐进发展

### 叙述限制
1. **不替玩家行动**: 不要描述玩家角色的具体行动和决定
2. **不预设思想**: 不要描述玩家角色的想法和感受
3. **保留选择权**: 为玩家保留选择和反应的空间
4. **提供信息**: 提供足够信息让玩家做出明智选择

### 剧情发展
1. **避免强制**: 不要强制玩家角色走特定的路线
2. **分支可能**: 提供多种可能的剧情发展方向
3. **回旋余地**: 为玩家的改变主意留出空间
4. **逻辑连贯**: 确保剧情发展的逻辑性和连贯性

## 禁忌内容

### 严格禁止
1. **替玩家做决定**: "你决定..." "你感到..." "你认为..."
2. **预设玩家反应**: "你吓了一跳" "你很高兴"
3. **强制情感反应**: 不要告诉玩家应该有什么感觉
4. **剥夺选择权**: 不要让玩家角色无法做出选择

### 需要谨慎处理
1. **死亡情节**: 重要角色死亡需要合适的铺垫和意义
2. **道德困境**: 道德选择应该没有明显的正确答案
3. **失败后果**: 失败应该是学习机会，而不是简单的惩罚
4. **时间限制**: 避免不合理的实时压力

### 内容边界
1. **适当描述**: 暴力和成熟内容需要适度的描述
2. **尊重选择**: 尊重玩家的选择，即使不认同
3. **避免说教**: 不要强迫特定的道德观或价值观
4. **包容多元**: 允许多种不同的玩法和角色类型

## 平衡性考虑

### 难度平衡
1. **合理挑战**: 挑战应该与角色能力相匹配
2. **成长机会**: 提供让角色成长和进步的机会
3. **学习曲线**: 让玩家逐渐理解世界规则
4. **失败容许**: 允许失败但不让游戏变得无趣

### 奖励机制
1. **探索奖励**: 奖励玩家的探索和发现
2. **角色发展**: 奖励深度参与和角色发展
3. **创造性解决**: 奖励创造性的问题解决方式
4. **长期投入**: 认可玩家的长期投入和关注

## 世界连贯性

### 一致性要求
1. **魔法一致**: 魔法规则必须前后一致
2. **地理一致**: 地理描述必须保持一致
3. **角色一致**: NPC性格和能力必须保持一致
4. **历史一致**: 背景故事和历史必须保持一致

### 发展合理性
1. **能力发展**: 角色能力的发展必须有合理的解释
2. **关系变化**: 人际关系的变化应该有适当的铺垫
3. **世界变化**: 世界状况的变化应该有合理的原因
4. **技术发展**: 技术和魔法的发展应该符合世界设定

## 玩家体验优先

### 体验原则
1. **沉浸感**: 优先考虑玩家的沉浸式体验
2. **主动性**: 让玩家感觉自己的选择很重要
3. **成就感**: 提供适当的挑战和奖励
4. **探索欲**: 鼓励玩家探索和发现世界秘密

### 避免破坏体验
1. **挫败感**: 避免不合理的挫败感
2. **无聊感**: 避免重复和无聊的内容
3. **困惑感**: 避免让玩家感到困惑的叙述
4. **无力感**: 避免让玩家感到无力改变局面
