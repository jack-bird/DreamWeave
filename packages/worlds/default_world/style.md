# 默认世界观文风指南

## 叙述风格

### 基本原则
- **第二人称视角**: 使用"你"来称呼玩家角色，增强代入感
- **展示而非讲述**: 通过行动和对话展现情节，而不是直接告知结果
- **感官描写**: 调动视觉、听觉、嗅觉、触觉等多重感官
- **节奏控制**: 在紧张场景加快节奏，在平静场景放慢描述

### 语言特色
- **避免现代词汇**: 不使用现代科技词汇和过于口语化的表达
- **保留魔法色彩**: 使用符合奇幻世界的表达方式
- **保持氛围感**: 根据场景调整语言风格（恐怖、温馨、紧张等）

## 角色对话风格

### 主角对话
- **简洁有力**: 玩家的输入通常转化为直接、有效的行动
- **体现性格**: 根据玩家选择展现不同的性格特质
- **避免冗长**: 不要替玩家说太多话，保持玩家的主动性

### NPC对话
- **个性鲜明**: 每个NPC都有独特的说话方式和用词习惯
- **信息传递**: 通过对话自然传达世界观和剧情信息
- **情感真实**: 让NPC的反应符合其性格和处境

## 场景描述技巧

### 环境描写
```
好的例子：
"古堡的石墙在火把光芒下投下长长的阴影，空气中弥漫着古老灰尘和淡淡霉味。远处传来滴水声，在这个寂静的空间里显得格外清晰。"

不好的例子：
"这是一个古老的城堡，看起来很恐怖，里面有很多东西。"
```

### 氛围营造
- **光影变化**: 利用光线和阴影创造氛围
- **声音效果**: 环境声音增强沉浸感
- **温度感受**: 描述温度变化增加真实感
- **时间流逝**: 通过环境变化体现时间推进

## 冲突和紧张感构建

### 逐步升级
1. **平静开始**: 正常的环境描述
2. **异常迹象**: 微妙的不对劲感觉
3. **紧张升级**: 明显的威胁出现
4. **高潮时刻**: 直接的冲突或危机
5. **后果处理**: 行动结果和新的情况

### 选择的重要性
- **明确后果**: 让玩家了解选择的可能结果
- **道德困境**: 提供没有完美答案的困难选择
- **长远影响**: 小的今天可能影响未来的情节

## 情感共鸣技巧

### 情感层次
- **表层情感**: 直接的情绪反应（恐惧、愤怒、喜悦）
- **深层情感**: 复杂的情感体验（内疚、希望、失望）
- **情感转变**: 角色情感的成长和变化

### 沉浸感营造
- **身体感觉**: 描述身体反应增强真实感
- **思维活动**: 展现角色内心思考和疑虑
- **情感共鸣**: 让玩家感受到角色的情感状态

## 避免的写作陷阱

### 常见问题
- **替玩家决定**: 不要描述玩家的思想和感受，让玩家自己体验
- **过度解释**: 相信玩家的理解能力，不要过度解释背景
- **节奏失控**: 避免过长或过短的段落
- **逻辑漏洞**: 确保剧情发展的合理性和连贯性
- **角色OOC**: 保持角色行为的一致性和合理性

### 质量检查要点
- **是否符合第二人称视角**
- **是否替玩家做了太多决定**
- **场景描述是否生动具体**
- **角色对话是否符合其性格**
- **世界观是否保持一致**
- **是否存在现代词汇或表达**

## 特殊场景处理

### 战斗场景
- **动作清晰**: 确保战斗动作的描述清晰易懂
- **后果明确**: 明确描述战斗的结果和影响
- **策略重要**: 体现策略和智谋的重要性

### 探索场景
- **发现过程**: 让玩家体验探索的乐趣
- **细节丰富**: 提供足够的环境细节供玩家发现
- **奖励合理**: 根据探索深度给予适当的奖励

### 社交场景
- **对话自然**: 让NPC的对话更加自然和真实
- **选择多样**: 提供多种社交策略和应对方式
- **关系发展**: 让玩家角色与NPC的关系逐步发展
