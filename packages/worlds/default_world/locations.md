# 默认世界观地点设定

## 主要地点

### 黑夜古堡
一个神秘的古堡，是许多冒险的起点。

**古堡大门**
- 巨大的铁木门，上面布满了岁月痕迹
- 门上雕刻着复杂的符文，似乎有魔法保护
- 两侧有石像鬼雕像，在特定时间会"活"过来
- 夜晚时分，古堡显得格外阴森

**古堡门厅**
- 进入大门后的第一个大厅
- 高高的天花板，挂着古老的画像
- 中央有一个巨大的水晶吊灯
- 多扇门通向古堡的不同区域
- 墙上挂满了武器和铠甲

**图书室**
- 收藏着大量古老的书籍和卷轴
- 有一些书籍可能包含重要的魔法知识
- 图书管理员是一个神秘的幽灵
- 某些书籍可能被诅咒或保护

**地牢**
- 古堡的地下区域
- 曾经关押过重要的囚犯
- 现在可能还有生物在这里活动
- 传说有秘密通道连接到外面

### 中央王国
曾经的魔法帝国中心区域。

**王城**
- 政治权力中心
- 国王和贵族居住的地方
- 有精美的宫殿和花园
- 经常有政治活动和宴会

**商业区**
- 繁华的市场
- 各种商人和工匠
- 可以买到装备和物品
- 也是信息交流的中心

**魔法师协会**
- 魔法研究和学习的中心
- 有大型图书馆
- 提供魔法训练和服务
- 需要会员身份才能进入

**贫民窟**
- 城市中的贫困区域
- 犯罪和危险较多
- 但也有有用的信息和联系
- 一些地下组织的活动场所

### 东方森林
神秘的精灵领地。

**森林边缘**
- 安全区域的边界
- 有冒险者营地
- 是进入森林的起点
- 有基本的补给和服务

**精灵村庄**
- 精灵的主要居住地
- 与自然和谐的建筑
- 精灵长老提供智慧建议
- 可以学习自然魔法

**古老神庙**
- 森林深处的古代遗迹
- 传说有强大的魔法源泉
- 被魔法生物守护
- 有复杂的谜题和陷阱

### 西方山脉
矮人王国的山地要塞。

**山门要塞**
- 进入矮人领地的关口
- 有强大的防御工事
- 矮人战士把守
- 需要通行证才能通过

**地下城市**
- 矮人的主要居住地
- 精美的地下建筑
- 有大型工坊和市场
- 可以定制装备和武器

**古代矿坑**
- 旧时的矿区
- 可能还有稀有矿石
- 但也有危险生物
- 传说有古代矮人宝藏

### 南方海岸
繁荣的贸易城市。

**海港城市**
- 最大的贸易港口
- 来自世界各地的商船
- 有丰富的商品和信息
- 但也有海盗威胁

**海盗岛**
- 海盗的藏身之处
- 有危险的罪犯
- 但也有宝藏和秘密
- 需要特殊方式才能到达

**海底遗迹**
- 海底的古代建筑
- 传说有失落的文明
- 需要潜水或魔法才能探索
- 有神秘的海底生物

## 特殊地点

### 传送门遗迹
- 古老的传送门网络
- 大部分已经损坏
- 某些可能还能使用
- 连接世界的不同区域

### 龙谷
- 巨龙的栖息地
- 极度危险的区域
- 但有龙族宝藏
- 只有勇敢者才敢前往

### 魔法森林
- 充满魔法能量的森林
- 有神奇的各种生物
- 植物有魔法特性
- 时间流逝可能不同

### 战争遗迹
- 古代战场的遗迹
- 仍然有战斗的痕迹
- 可能有幽灵和亡灵
- 有古代武器和装备

## 地点互动原则

### 探索原则
1. **渐进探索**: 新地点应该逐步开放
2. **准备重要**: 某些地点需要适当准备才能安全探索
3. **多样性**: 不同地点有不同特色和挑战
4. **连通性**: 地点之间有合理的连接方式

### 环境互动
1. **细节丰富**: 每个地点都有详细的描述
2. **隐藏要素**: 有隐藏的东西等待发现
3. **变化可能**: 地点可能因为玩家行动而变化
4. **历史深度**: 每个地点都有自己的历史和故事

### 氛围营造
1. **感官描述**: 视觉、听觉、嗅觉等多重感官
2. **时间变化**: 不同时间有不同的氛围
3. **天气影响**: 天气影响环境和游戏体验
4. **季节变化**: 季节带来不同的景色和挑战
